Tabula

Description:

  • Tabula.js models and runs the Roman board game Tabula.

    The module exposes the game state, legal move calculation, turn progression, captures, bar re-entry, bearing off, and win detection for the standard 24-point Tabula board.

Example
let game = Tabula.createGame();

    game = Tabula.movePiece(game, { from, to });

(static) getLegalMoves(state, pointIndex) → {Array.<(number|Tabula.Location)>}

Description:

  • Return every legal destination reachable from a source point using the remaining dice in any valid order. Intermediate steps must also be legal, and bar re-entry takes priority over normal board movement.

    Domain rules applied:

    • If the current player has pieces on the bar, only BAR can be used as the source.
    • Destinations occupied by two or more opponent pieces are blocked.
    • A destination can be reached via a multi-die path only when each intermediate step is legal.
    • OFF is included only when bearing-off rules are satisfied.

    If move.dicePath is omitted, any legal dice sequence for that source/destination pair is accepted.

(static) isTurnComplete(state) → {boolean}

Description:

  • Check whether the active turn is finished.

    A turn is complete when no dice remain, or when remaining dice cannot produce any legal move from required sources.

    Bar priority is enforced: if the active player has pieces on the bar, only bar re-entry is considered when determining completion.

(static) movePiece(state, move) → {Tabula.GameState}

Description:

  • Execute one legal Tabula move and return the next game state.

    Domain effects:

    • Captures a single opposing piece by sending it to the bar.
    • Removes only the dice values consumed by this move.
    • Clears selectedPoint after the move resolves.
    • Sets winner when the active player has borne off all 15 pieces.

    If the move is not legal, the original state is returned unchanged.
